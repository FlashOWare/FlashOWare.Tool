# flashoware interceptor check

Available since [v1.0.0].

## Synopsis
```console
flashoware interceptor check

flashoware interceptor check -?|-h|--help
```

## Description

Show whether experimental interceptors are supported or not by discovered .NET SDKs.

## Options

- `-?|-h|--help`  
Show help and usage information.
